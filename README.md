# SleepHealthAnalysis
DataSc101 project

## Описание
Анализ данных о качестве сна и здоровье людей различных профессий.

## Содержание
- 'SleepHealthAnalysis.ipynb' - Jupyter Notebook с полным анализом
- 'Sleep_health_and_lifestyle_dataset.csv' - Исходный датасет
- 'sleep_data_clean.csv' - Обработанный датасет
- ссылка на Yandex Datalens

## Структура анализа
### Часть 1: Python анализ
- Загрузка и обзор данных
- Обработка пропусков и дубликатов
- Описательная статистика и корреляции
- 6 визуализаций (гистограммы, графики, boxplot, heatmap)
- Группировки и агрегации по профессиям и полу

### Часть 2: Yandex DataLens
- Интерактивный дашборд
- 3 чарта с фильтрами и селекторами
